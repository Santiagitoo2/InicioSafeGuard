🚀 Características Principales

🔐

Autenticación Segura

Inicio de sesión con Google que detecta automáticamente tu dispositivo (iPhone, Android, PC) usando Redirect o Popup según sea necesario.

🚨

Botón SOS

Alerta de emergencia con un solo toque. Envía tu ubicación exacta a todos tus contactos de emergencia por WhatsApp y Email.

🗺️

Mapa Interactivo

Ubicación en tiempo real, búsqueda de hospitales, comisarías y rutas de navegación. Adaptado para cada dispositivo.

🤖

Asistente IA

Chatbot inteligente con respuestas para emergencias médicas, incendios, accidentes y más. Comandos de voz disponibles.

🕵️

Modo Discreto

Calculadora falsa que oculta la app. Ideal para situaciones de riesgo. Incluye botón SOS oculto dentro.

📞

Llamada Simulada

Simula una conversación telefónica realista para disuadir agresores. Llamada real al 911 también disponible.

// Autenticación multi-dispositivo
async function loginWithGoogle() {
    const isMobile = /iPhone|iPad|Android/i.test(navigator.userAgent);
    
    if (isMobile) {
        // En móvil usar Redirect (más confiable)
        await signInWithRedirect(auth, provider);
    } else {
        // En PC usar Popup (mejor experiencia)
        const result = await signInWithPopup(auth, provider);
    }
}

// Envío de WhatsApp multi-plataforma
function sendWhatsApp(phone, message) {
    if (isIOS) {
        window.location.href = `whatsapp://send?phone=${phone}&text=${message}`;
    } else {
        window.open(`https://wa.me/${phone}?text=${message}`, '_blank');
    }
}
